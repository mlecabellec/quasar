# Classes

This directory contains the implementation details of the classes of the Quasar project.

It is organized by namespaces and shall contain one directory per namespace.

In each namespace, it shall contain one file per class. 

Each file associated to a class shall contain:
- [IMPL-CLASSES-001] The class textual description, including its purpose, its responsibilities, its invariants, its pre-conditions, its post-conditions, and its constraints.
- [IMPL-CLASSES-002] The list of all methods of the class, with a short description of each method.
- [IMPL-CLASSES-003] The list of all attributes of the class, with a short description of each attribute.
- [IMPL-CLASSES-004] The list of all relations to other classes, with a short description of each relation.
- [IMPL-CLASSES-005] The list of all dependencies to other classes, with a short description of each dependency.
- [IMPL-CLASSES-006] The list of all tests of the class, with a short description of each test.
- [IMPL-CLASSES-007] The list of all examples of the class, with a short description of each example.
- [IMPL-CLASSES-008] A plantuml class diagram of the class.
