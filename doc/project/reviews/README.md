# Project Reviews

This directory is organized by date and contains the reviews of the project. Directory is structured by year, then by month.

Each review is a markdown file with the following structure:

- [REVIEW-001.1] Review reference structured as REVIEW-xxxxx where xxxxx is a 5 digit number.
- [REVIEW-001.2] Review title, description and purpose.
- [REVIEW-001.3] List of reviewed  code.
- [REVIEW-001.4] List of reviewed features and constaints.
- [REVIEW-001.5] List of issues found during review.
- [REVIEW-001.6] List of issues resolved during review.
- [REVIEW-001.7] List of issues not resolved during review.
- [REVIEW-001.8] List of issues to be reviewed in the future.
